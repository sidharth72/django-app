Python


Python is incredibly one of the efficient programming language comes with simple syntax and concepts. Your code in python is very easy to read and debug. Developers use Python for many purposes like Machine Learning and AI , Web development , Game development , GUI apps , automation etc. Python community is really vast and it is very important to have a community support because programming isn't a solitary pursuit. Anyway python is considered as the programming language of fucture.

Features:

1. Python is a interpreted programming language

An interpreter is a computer program that directly executes instructions written in a programming without requiring them previously to have been compiled into a machine language program. It is easy to spot errors when using an interpreted language

2. High level programming language(general purpose)

A high-level programming language is a programming language with strong abstraction from the details of the computer. Unlike low-level programming languages, it may use natural language elements, be easier to use, or may automate significant areas of computing systems (e.g. memory management), making the process of developing a program simpler and more understandable than when using a lower-level language.

3. Object-oriented

'Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data. Data is in the form of fields also known as attributes or properties, and code, in the form of procedures known as methods. A feature of objects is that an object's own procedures can access and often modify the data fields of itself.

4. Free and Open source

Open source is source code that is made freely available through out the internet with possible modification. Products include permission to use the source code, design documents, or content of the product. It most commonly refers to the open-source model, in which open-source software or other products are released under an open-source license.

5. Python is portable

Python is a portable language , ie it can run on any operating system like windows , linux , macos etc without actually changing the source code.

6. Easy to write and impliment code

The syntax of python is really not a task to learn , it is quite simple to learn for any person who is not from programming background.

7. Bunch of standard library

Python comes with lot of standard library in itself makes the workflow simpler.

8. Easy to debug code

Since python syntax is very easy to learn it is also very easy to debug code written in python.

Applications:

1. Artificial Intelligence & Machine Learning

ML and AI is the fucture of our technological progress. Some of the libraries like 'Scikit-Learn' , 'Pandas' , 'Numpy' etc are widely used in case of ML , AI and also to create neural networks.

2. Data science

Data science is a booming technology in the present and for the fucture. In python, libraries like 'matplotlib','seaborn' are used to visualise data.

3. Web development

Python comes with very popular web development frameworks like 'Django' and 'Flask' which are very popular among developers.

4. Game development

Libraries like 'Pygame' is used for Game development

5. Desktop GUI

Python offers Desktop GUI apps by using libraries like Tkinter , PyQt , PyGui , Kivy etc.

6. Automation

Since python is very simple , it can be used for automating huge tasks with little amount of code.

7. Ethical Hacking

Due to lots of libraries and frameworks python is not excluded from ethical hacking and pentesting.

Cons:

1. Speed - Python is an interpreted programming language thats why it is slow as compared to Java or C.

2. Memory usage - Python consumes lots of memory.

3. Runtime Errors - Python is dynamically typed and the datatypes can change at any time. This leads to runtime errors.
